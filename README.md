# POC DPD VOC - Sentiment Analysis Dashboard

A simple Vue.js dashboard for sentiment analysis over agent-to-customer conversations.

## Stack

- Vue.js 3
- Tailwind CSS
- Vite

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

```
├── index.html          # Entry HTML file
├── src/
│   ├── main.js        # Vue app entry point
│   ├── App.vue        # Main app component
│   └── style.css      # Global styles with Tailwind
├── package.json
├── vite.config.js
└── tailwind.config.js
```

## Features

- Left sidebar navigation
- Top bar with filters (organizations, teams, users, date range)
- Grid layout for future graphs
- Responsive design with Tailwind CSS
